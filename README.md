# Курсовая работа  
# Разработка системы анализа и прогнозирования исхода матча в Dota 2 на основе данных со стадии выбора героев  
Проект демонстрирует полный ML-пайплайн: от сбора данных и разведочного анализа до feature engineering, обучения моделей и интерпретации результатов.  
## Цель
Построить модель, которая до начала матча предсказывает вероятность победы команды Radiant, используя только информацию о драфте и базовую игровую статистику.

## Данные
* ~140 000 матчей
* Источник: OpenDota API
* Основные признаки:  
    * выбранные герои Radiant и Dire  
    * средний рейтинг игроков (avg_rank_tier)  
    * бинарный таргет (radiant_win)

Данные были очищены, валидированы и приведены к единому формату.

## Разведочный анализ данных
Проведена проверка качества данных, анализ распределения таргета и рейтинга,  
исследованы популярность и винрейты героев, а также базовые закономерности драфта,  
использованные далее при построении моделей.  

***Выводы EDA***  

- Данные сбалансированы и подходят для бинарной классификации.
- Состав команды оказывает влияние на результат матча.
- Популярность героев не равна их эффективности.
- Для моделирования требуются нелинейные методы.

## Feature Engineering
Реализованы два подхода кодирования признаков, ориентированные на разные типы моделей:

***One-Hot Encoding по героям***  
Используется для:  
* Logistic Regression  
* Random Forest  
Каждый герой кодируется как:  
* 1 — присутствует в Radiant  
* -1 — присутствует в Dire  
* 0 — отсутствует  

Подход хорошо работает с линейными и ансамблевыми моделями.

***One-Hot / categorical encoding по слотам***  
Используется для:  
* CatBoost  

Герои передаются в виде слотов (radiant_slot1, dire_slot3, …).  

Хотя порядок героев не соответствует игровым ролям, CatBoost эффективно извлекает взаимодействия и синергии между категориями, игнорируя порядок.

***Дополнительные признаки***  
* популярность героев;  
* агрегированные характеристики команд;  
* синергия выбранных героев.

## Модели и результаты

| Модель               | Accuracy      | ROC-AUC  |
| ---------------------| ------------- |----------|
| Logistic Regression  | 0.57          | 0.59     |
| Random Forest        | 0.60          | 0.63     |
| CatBoost             | 0.61          | 0.65     |

CatBoost показал лучшие результаты, благодаря способности моделировать сложные взаимодействия между героями.  

## Инсайты

* Драфт оказывает существенное влияние на исход матча, но не является единственным фактором.
* Прогноз по драфту принципиально ограничен влиянием исполнения игроков, стратегии и патчей.
* Даже без информации о ходе игры модель достигает ~61% accuracy, что значительно выше случайного угадывания.
* Результаты находятся близко к теоретическому пределу прогнозируемости драфта (≈58–70%), что соответствует исследованиям в области MOBA-аналитики.
